# font

- font 속성은 글자의 폰트를 정의하는 속성인데, 여러 기능을 동시에 가지고 있는 속성.
- 정확히 말하면 아래 6개의 세부적인 글꼴 관련 속성을 font라는 하나의 속성에 한번에 쓸 수 있다는 소리

1. `font-style` : 이탤릭체 등의 글꼴의 스타일 지정
2. `font-weight` : 글자 두께
3. `font-variant` : 글꼴 변형 (소문자를 대문자로 바꾸는 등의)
4. `font-size` : 글자 크기
5. `line-height` : 줄 간격
6. `font-family` : 글꼴 (굴림, 돋움, …)
