# 반응형 웹의 3요소

<hr/>

> **반응형 웹의 3요소**

1. 유연한 그리드: 이는 모든 레이아웃 요소가 상대적인 단위로 크기가 지정되는 방식을 의미한다. 예를 들어, 픽셀(px) 대신 백분율(%)을 사용하여 요소의 너비를 지정하면, 부모 요소의 크기에 상대적으로 자식 요소의 크기가 조절돼서 이렇게 하면 화면 크기가 변해도 레이아웃이 유지되며, 다양한 화면 크기에 대응할 수 있다.
2. 유연한 이미지/미디어: 이미지나 비디오 등의 미디어도 상대적인 단위를 사용하여 사이즈를 조절돼서 화면 크기가 변화해도 이미지나 비디오 등이 오버플로우 되거나 확대/축소 되는 것을 방지하고, 올바른 비율로 표시될 수 있게 된다.
3. 미디어 쿼리: CSS3에서 도입된 미디어 쿼리는 특정 조건에만 CSS 스타일을 적용하는 기능으로 이를 통해 다양한 디스플레이 타입과 환경에 맞춰 서로 다른 스타일 규칙을 적용이 가능
